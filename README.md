# 🏄 Projeto Soul Surf - Site Oficial

Site oficial do Projeto Soul Surf - Um projeto social que usa o surf como ferramenta de inclusão, educação e transformação de vidas em Itanhaém/SP.

## 📋 Sobre

O Projeto Soul Surf oferece diversas iniciativas:
- **Esporte & Surf**: Pega Esta Onda, Surf Adapt, CT Soul Surf, Garotas nas Ondas
- **Educação & Capacitação**: Soul Surf Digitais, Inglês, Design Gráfico
- **Meio Ambiente**: Educação Ambiental, Mutirões de Limpeza
- **Social & Saúde**: Oficinas Socioeducativas, Ações Solidárias

## 🌐 Publicação no GitHub Pages

Para publicar este site no GitHub Pages:

### 1. Criar repositório no GitHub
- Vá para [github.com/new](https://github.com/new)
- Nome: `soul-surf-site` (ou seu-usuario.github.io para site principal da conta)
- Descrição: "Site oficial do Projeto Soul Surf"
- Público
- Clique em "Create repository"

### 2. Adicionar Remote e Push

```bash
cd /home/michelle/soul-surf-site

# Adicionar remote (substitua USERNAME pelo seu usuário GitHub)
git remote add origin https://github.com/USERNAME/soul-surf-site.git

# Renomear branch para main (opcional)
git branch -M main

# Push do código
git push -u origin main
```

### 3. Ativar GitHub Pages

- Vá para Settings → Pages
- Source: Deploy from a branch
- Branch: main
- Folder: / (root)
- Clique em Save

Seu site estará em: `https://USERNAME.github.io/soul-surf-site/`

## 🛠️ Desenvolvimento Local
